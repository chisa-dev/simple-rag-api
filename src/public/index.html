RAG System API

A simple but functional RAG (Retrieval Augmented Generation) system API for demonstration purposes.

API Documentation

This API provides endpoints for document indexing and RAG-based chat functionality.

GET /api/status: Check the status of the RAG system
POST /api/rag/index: Upload and index a document (supports PDF, DOCX, PPT, images, and text files)
POST /api/rag/chat: Generate a chat response using the RAG approach (only requires a query parameter)
View Swagger Documentation

Architecture Overview

Document Processor

Extracts text from documents, splits into chunks, and generates embeddings.

Vector Store

Stores and retrieves document embeddings using Qdrant.

Content Generator

Generates responses using OpenAI based on retrieved context.

API Layer

RESTful endpoints with Swagger documentation.
